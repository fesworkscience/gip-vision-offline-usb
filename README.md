# GIP Vision Offline USB

Локальный офлайн-конвертер `IFC -> USDZ` для подготовки моделей к просмотру в приложении **GIP VISION** (вкладка **Offline**).

## Что делает сервис
- Запускает локальный веб-интерфейс на `http://127.0.0.1:8765`
- Принимает `.ifc` файл
- Конвертирует модель в `.usdz`
- Автоматически сохраняет результат в папку `usdz` на носителе

## Быстрый запуск
### macOS
```bash
cd /Volumes/gip_vision/gip-vision-offline-usb-macos-arm64
python3 start.py
```
Также можно запускать `start.command`.

### Windows
Запустите `run.bat`.

## Где лежат файлы
- `ifc/` — входные IFC
- `usdz/` — результаты конвертации USDZ
- `config/workspace/jobs/` — служебные данные задач

## Открытость и безопасность
- Репозиторий открыт: https://github.com/fesworkscience/gip-vision-offline-usb
- Исходный код и история изменений доступны в GitHub, поэтому поведение сборки можно проверить.
- Конвертация выполняется локально на вашей машине, без отправки моделей в облако.
- В рантайме сборки включена блокировка сетевого доступа для процесса конвертации.
- При запуске сервера процесс **не имеет доступа в интернет**, поэтому никакие пользовательские данные не отправляются на внешние серверы.

## Релизы
- Актуальные релизы: https://github.com/fesworkscience/gip-vision-offline-usb/releases
- Версия текущей сборки хранится в `config/version.yaml` и отображается в интерфейсе.
