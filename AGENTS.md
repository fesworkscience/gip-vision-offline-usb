# AGENTS

## Правила для релизов (обязательно)
- В каждом GitHub Release обязательно заполнять описание для пользователя с шагами установки и запуска сервера.
- В описании релиза обязательно указывать:
1. Скачать сборку из Releases.
2. Распаковать архив в корень флеш-носителя.
3. macOS: запустить `start.command` или `python3 start.py`.
4. Windows: запустить `run.bat`.
5. Открыть `http://127.0.0.1:8765`, если браузер не открылся автоматически.
- В каждом релизе обязательно добавлять полный список коммитов между предыдущим и текущим тегом.
- Формат диапазона: `<prev_tag>..<current_tag>`.
